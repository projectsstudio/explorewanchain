Gwan

Gwan

Gwan is the GO implementation of Wanchain and the command line interface for running running a full Wanchain node. Gwan allows users to interact with the Wanchain network, create accounts, transfer funds, deploy and interact with contracts. To learn more, check out the Github page or navigate to our developer portal.
